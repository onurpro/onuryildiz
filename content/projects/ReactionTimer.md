**Role:** Developer
**Course:** Academic Project — Sep 2024 to Dec 2024

## Project Overview

Designed and implemented a reaction-based game using **VHSIC Hardware Description Language (VHDL)**, simulating hardware behavior in **Quartus Prime** and **ModelSim**.

## Design Details

### Finite State Machine
Developed an **FSM (Finite State Machine)** to manage game states, including:
- Player turns and input handling
- Random stimuli generation
- Winner determination logic

### I/O Integration
- **LED indicators** to distinguish between players
- **Seven-segment displays** to show game results and reaction times

### Reset Functionality
Designed both **soft and hard reset** functionality:
- **Soft reset** — resets the current session for a new round
- **Hard reset** — system-wide reset for full re-initialization

## Key Learnings

This project gave me hands-on experience with digital design using VHDL, FSM-based system design, and hardware simulation workflows with industry-standard tools like Quartus Prime and ModelSim.
